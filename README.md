# Black Cat 活动分享社交平台

一个活动分享的社交平台，访客可以根据一些过滤条件来浏览活动、点赞或加入活动，以及查看其他人的资料与发起的活动。

## 技术选型

技术栈：React Native + mbox + typescript
多语言：i18n-js
代码检查：gts

## 安装
npm install

## 开发构建
npm run start

## 基本功能
1、未登录时需要登录后查看活动，支持切换语言
